# FORGE To Skills Mapping

This reference maps the current FORGE repository concepts into a bundled skill product.

## Move Into Skills

- execution workflow
- hard stop behavior
- critique workflow
- security review procedure
- evaluation workflow
- memory read/write procedure

## Keep As Repo-Local State

- active mode and execution style
- collaboration mode and branch policy
- task list
- project architecture constraints
- project-specific evaluation rules
- project memory entries
- team coordination rules
- tool-specific `forge.yaml`

## Keep As External Enforcement

- commit hooks
- CI scripts
- branch protection rules
- release and sync scripts

## Practical Outcome

The agent loads stable FORGE logic from the skill pack and reads only the repo-local data required for the current task.
