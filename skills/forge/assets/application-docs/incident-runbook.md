## Purpose

Provide immediate response steps for common incidents affecting this tool.

## Trigger Conditions

- Service unavailable
- Degraded performance
- Failed job or pipeline
- Auth or connectivity failure

## Immediate Actions

1. Confirm the symptom and blast radius.
2. Check dashboards, logs, and alerts.
3. Validate recent changes or deployments.
4. Apply the appropriate mitigation.
5. Escalate if recovery is not immediate.
